STRAIGHT LEND
Straight Lend, a platform that simulates
the P2P lending process.
Users can play the role of a borrower or
a lender and go through each loan application step interactively.
THE FRONTEND IS BUILT WITH HTML, CSS, AND JAVASCRIPT.
WHILE THE BACKEND USES NODE.JS AND EXPRESS.JS FOR SERVER-SIDE OPERATIONS.
MYSQL IS USED TO MANAGE USER AND TRANSACTION DATA.
Power BI dashboard
This project is a Power BI dashboard that analyzes global video game sales.
It provides insights into top-selling platforms, popular game genres, sales trends by year, and game distribution by platform and rating, with interactive filters for deeper exploration.
