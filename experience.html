STRAIGHT LEND

FULL-STACK P2P LENDING SIMULATION PLATFORM

Straight Lend is a web application designed to demonstrate a peer-to-peer (P2P) lending process in a simulated, academic environment without actual transactions.

My main role involved implementing core system functions including frontend development, backend API development, and database connectivity. I also collaborated on ER diagram design and UI/UX design.

The platform features user sign-up for both lenders and borrowers, loan request creation, identity document uploads for verification, approval tracking, admin dashboard for user validation, interest rate enforcement (5% annually), and transaction monitoring under Thai financial regulations.

KEY HIGHLIGHTS:

  • Role-based authentication (Borrower / Lender / Admin)
  • End-to-end loan request and offering workflows
  • Document submission and admin verification process
  • Interest rate capped at 5% annually in compliance with regulations
  • Real-time relational data storage powered by MySQL
  • Administrative dashboard with validation and risk metrics

TECH STACK:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Node.js, Express.js
  • Database: MySQL (relational schema)
  • Collaboration: Co-designed UI and ER Diagram

FAKE NEWS DETECTION

MACHINE LEARNING & NLP

The system uses BERT (bert-base-uncased) fine-tuned via transfer learning for binary classification of real vs fake news. I handled the entire development pipeline, including data preparation, model training with NLP techniques, and building a Flask web interface. The model was implemented using PyTorch and Hugging Face Transformers, achieving over 90% accuracy.

The goal is to promote awareness and help users verify news content in real-time before spreading misinformation.
